## TECHNOLOGY

 Created using `HTML`, `SCSS`, `JAVASCRIPT`, `NODE.JS`, `REACT`, `VITE`, `EXPRESS`, `MONGODB`
 
 <br>
 
 ### How to start backend
 
 ```bash
 git clone https://github.com/tathagatamishra/MyBlog.git
 cd server
 npm i
 npm run dev
 ```
 
 ### How to start frontend
 
 ```bash
 git clone https://github.com/tathagatamishra/MyBlog.git
 cd client
 npm i
 npm run dev
 ```
